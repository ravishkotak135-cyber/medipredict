# Disease Prediction System (Medipredict)

This is my semester 8 project where I built a disease prediction system using machine learning and a simple web interface.

The main idea of the project is to take user input (like health conditions) and predict possible diseases.

## Tech Used

- Python
- Flask
- HTML, CSS
- Machine Learning (Random Forest)

## How it works

The user enters health-related data in the form.  
That data is sent to the backend using Flask API.  
The trained model then predicts the disease and sends the result back to the UI.

## Features

- Simple and clean UI
- Fast prediction
- Backend API integration
- Basic chatbot (for interaction)
